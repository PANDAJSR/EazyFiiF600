# EazyFiiF600

# 简介
这是一个小鸟飞飞F600的第三方编程软件，兼容官方格式，具备2D、3D路径实时预览/鼠标画路径生成代码/问题（撞杆/科目完成）检测等功能，使用 Electron 开发

# 如何使用
## 方法一：从源代码运行
``` bash
git clone https://github.com/PANDAJSR/EazyFiiF600.git

pnpm install
pnpm dev:electron # 启动vite开发服务器并运行electron
```
## 方法二：使用打包版本
- 前往 [Action 中的打包工作流](https://github.com/PANDAJSR/EazyFiiF600/actions/workflows/build-desktop-manual.yml)自行下载相应平台的可执行文件
