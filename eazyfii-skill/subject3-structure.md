# 科目③结构说明

## 核心要点

科目③的圆环**夹在两个竖杆中间**，无人机只需：

> **从两杆中间按照圈圆心的高度穿过去即可**

## 结构示意

```
    竖杆1          竖杆2
      |      ◯      |
      |    (圆环)   |
      |             |
     杆子间距约1米，圈在中间
```

## 编程提示

1. **水平位置**：对准两杆中间（X/Y坐标的中间值）
2. **高度**：圈中心离地高度由现场题卡给出，飞到这个高度即可
3. **无需转向**：直线穿过，保持机头朝向飞行方向
4. **安全余量**：建议飞行高度略高于圈中心（如圈中心1.2米，可飞1.3米）

## 与科目②的区别

| 科目 | 结构 | 核心动作 |
|------|------|----------|
| ② | 横杆在两杆之间 | 从横杆下方飞过 |
| ③ | 圆环在两杆之间 | 从圆环中心穿过 |

## 示例坐标（仅供参考）

假设现场测量：
- 竖杆1位置：(120, 120)
- 竖杆2位置：(120, 220)
- 圈中心高度：1.2米（由题卡给出）

则穿越点：
- 水平位置：(120, 170) —— 两杆中间
- 高度：Z = 120（略高于1.2米，单位cm）
