# AGENTS

## 技术栈与主要包
- 项目类型：前端单页应用
- 构建工具：Vite
- 框架：React + TypeScript
- 组件库：Ant Design（`antd`）
- 包管理器：pnpm
- 代码规范：ESLint + TypeScript ESLint

## 代码文件规模约束
- 不要编写超过 400 行的代码文件。
- 需要在接近 400 行前就主动拆分文件，按模块与职责合理分配。
- 目标是提升可维护性，不要等到超过 400 行后才创建新文件。

## 提交流程约束
- 每次修改代码内容后，都必须提交并推送。
- Commit 消息必须使用中文，并遵循 Conventional Commits 规范。
